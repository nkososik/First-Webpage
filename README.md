## Description

_{This webpage was created in order to learn the basics of HTML webpage building, CSS styling, and Git Branching and Merging. It showcases the usage of both inline and block elements, CSS Text styles, hyperlinks, image references, as well as utilizing github's repositories to create branches for other developers to work on. }_
